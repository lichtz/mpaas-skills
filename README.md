## 🚀 快速开始

### 1. 选择平台

根据你的开发平台选择对应的快速接入指南：

- **Android** → 参考 `mpaas-android/SKILL.md`
- **iOS** → 参考 `mpaas-ios/SKILL.md`
- **鸿蒙** → 参考 `mpaas-harmony/SKILL.md`

### 2. 选择功能模块

根据业务需求选择功能模块：

- 需要API管理 → `mpaas-gateway`
- 需要用户登录 → `mpaas-auth`
- 需要离线支持 → `mpaas-sync`
- 需要数据统计 → `mpaas-analytics`
- 需要问题排查 → `mpaas-crash`
- 需要性能优化 → `mpaas-performance`
- 需要扫码功能 → `mpaas-scan`
- 需要H5加速 → `mpaas-offline`
- 需要热修复 → `mpaas-hotfix`
- 需要推送通知 → `mpaas-push`

## 📋 前置条件

所有模块都需要：

1. 阿里云账号
2. 已开通mPaaS产品
3. 对应平台的开发环境
