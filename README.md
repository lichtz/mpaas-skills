# mPaaS Skills Collection

mPaaS技能集合 - 包含Android、iOS、鸿蒙平台接入指南及核心功能模块。

## 快速开始

选择对应的平台或功能模块进行接入：

- `mpaas-harmony/` - 鸿蒙快速接入
- `mpaas-android/` - Android快速接入  
- `mpaas-ios/` - iOS快速接入
- `mpaas-backend/` - 后台服务

## 核心功能

- `mpaas-gateway/` - 移动网关
- `mpaas-auth/` - 用户认证
- `mpaas-sync/` - 数据同步
- `mpaas-analytics/` - 移动分析
- `mpaas-crash/` - 崩溃分析
- `mpaas-performance/` - 性能监控

## 增值功能

- `mpaas-scan/` - 统一扫码
- `mpaas-offline/` - 离线包
- `mpaas-hotfix/` - 热修复
- `mpaas-push/` - 消息推送

## 文档

详细接入指南请参考各模块的 SKILL.md 文件。
